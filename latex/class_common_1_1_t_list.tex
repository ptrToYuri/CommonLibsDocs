\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename T$>$\newline
class Common\+::\+TList$<$ T $>$}

Container representing Doubly linked list. 

\begin{DoxyWarning}{Warning}
Work in progress. Sentinel node is not implemented yet, so iterators do not work properly. There may be other bugs
\end{DoxyWarning}
Iterating over elements is slow, so you should use iterators that store position. Supports fixed-\/time add/removal of elements (and it is very fast).

\mbox{\hyperlink{class_common_1_1_t_list}{TList}} WILL have methods like indexing for compatibility, but they are supposed to be very slow because iteration is needed.

Exception policy\+: none (for now).

\begin{DoxyNote}{Note}
If the logical result of operation may not be determined, there are two methods\+: one that throws an exception (\char`\"{}\+Safe\char`\"{} prefix) and another one that does assertion in debug mode. Custom exception is \mbox{\hyperlink{class_common_1_1_c_out_of_range}{COut\+Of\+Range}}, derived from \mbox{\hyperlink{class_common_1_1_c_exception}{CException}}.
\end{DoxyNote}
