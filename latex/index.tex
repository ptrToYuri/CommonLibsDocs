\begin{DoxyEnumerate}
\item Create Visual Studio solution or use existing one 
\item Place this folder into your solution. I recommend using git submodules, because it allows to update lib easily\+: 
\begin{DoxyEnumerate}
\item {\ttfamily git init}, if you do not have repository yet 
\item {\ttfamily git submodule add \href{https://github.com/ptrToYuri/CommonLibs.git}{\texttt{ https\+://github.\+com/ptr\+To\+Yuri/\+Common\+Libs.\+git}}}
\end{DoxyEnumerate}
\item Right click on solution in Visual Studio -\/$>$ Add -\/$>$ Existing project -\/$>$ find and select \char`\"{}\+Common\+Libs.\+vcxproj\char`\"{} in the file explorer 
\item Right click on project in which you want to use with this lib -\/$>$ Add -\/$>$ Reference -\/$>$ Select \char`\"{}\+Common\+Libs\char`\"{} -\/$>$ Ok
\end{DoxyEnumerate}\hypertarget{index_Update}{}\doxysection{Update}\label{index_Update}
